# Znečištění vzduchu

Scrape a vizualizace hodinových průměrů koncentrací NO2, O3, PM10 a SO2 z měřících stanic ČHMÚ.

Zatím nevydáno.

Zdroj dat: [JSON ČHMÚ](http://portal.chmi.cz/files/portal/docs/uoco/web_generator/aqindex_cze.json).
